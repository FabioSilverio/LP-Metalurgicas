# LP Metalurgicas

Landing page estatica da Pandora Tecnologia, focada em metalurgicas, serralherias e lojas de metais.

## Arquivos

- `index.html`: estrutura da pagina
- `styles.css`: layout, cores e responsividade
- `script.js`: animacoes e interacoes
- `assets/img/`: imagens e icones da pagina

## Publicacao

O repositorio esta preparado para publicar no GitHub Pages via GitHub Actions.
